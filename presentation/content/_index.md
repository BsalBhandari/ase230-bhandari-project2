# ASE230 Project 2 - Laravel REST API

## Project Overview

This project re-implements Project 1 APIs using Laravel framework with:
- **Laravel REST API** - Modern PHP framework with Eloquent ORM
- **Laravel Sanctum** - API token authentication
- **Docker Deployment** - Containerized application
- **Hugo Documentation** - Static site documentation
- **GitHub Actions** - Automatic deployment to GitHub.io

## Features

✅ Re-implemented all Project 1 endpoints  
✅ Docker containerization  
✅ Automated deployment scripts  
✅ Hugo documentation site  
✅ Automatic GitHub.io deployment
